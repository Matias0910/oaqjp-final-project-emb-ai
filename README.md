# Final Project: Emotion Detector

An AI-based web application developed using Python, Flask, and the Watson NLP library. This application analyzes text to detect emotional tones (anger, disgust, fear, joy, and sadness) and identifies the dominant emotion.

## Project Structure
* `EmotionDetection/`: Package containing the core logic and integration with Watson NLP.
* `server.py`: Flask server deployed for the web interface.
* `test_emotion_detection.py`: Unit tests to validate application performance.
